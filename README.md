# 🚕 AimGo
AimGo — это мобильное приложение такси, ориентированное на регионы и небольшие города, где крупные сервисы вроде Яндекс или inDrive работают неэффективно или экономически невыгодны.

Проект решает простую проблему:
в местах с низкой плотностью заказов и своими локальными тарифами нужен более гибкий и лёгкий сервис.

💡 Идея
В небольших городах и областях: 
* нет смысла использовать сложные алгоритмы ценообразования
* водители и пассажиры часто работают по фиксированным или локальным тарифам
* крупные агрегаторы берут слишком высокую комиссию

AimGo предлагает:
* простую модель поездок
* фиксированную цену (или локальные правила)
* минимальную инфраструктуру

⚙️ Технологии
*Backend*
Go (Golang)
REST API (Gin)
PostgreSQL

*Mobile*
Swift
SwiftUI

*Инфраструктура*
Docker
OpenStreetMap (OSM)

🧩 Основной функционал
* Создание поездки (passenger → driver)
* Принятие поездки водителем
* Начало и завершение поездки
* Статусы: pending → accepted → ongoing → done
* Работа с координатами (lat/lng)

🗺️ Почему OSM
Вместо коммерческих карт используется OpenStreetMap:
* бесплатный
* подходит для кастомных решений
* удобен для регионов с ограниченной поддержкой

🚀 Запуск проекта
*Backend*
```
go mod tidy
go run trip-service/main.go
```

Сервис поднимается на:
http://localhost:8005

База данных
Используется PostgreSQL:
```
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";
```

*Docker (опционально)*
```
docker-compose up --build
```
